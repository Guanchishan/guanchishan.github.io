颱風前夜的仙草凍

刊於 2019-08-29 | 改於: 2019-08-30 | 屬於 紀行 | | 閱覽:

傍晚時分，去城西湖畔的一家書屋，拿起一本科幻集子，買了杯芒果酸奶，佔了座。

學習完其提綱挈領的科幻文學簡史——兼引言，讀了頭四篇選文——《帕拉納曼科》《雨中哭喊》《冰凍的紅衣主教》《戀愛中的瑞秋》。想像力是頂好的，象徵也頗迷人。特別是如《帕》那篇一樣用力去「愛」一座城，真是想想就讓人受不了呢。回過神來，外頭已是雷聲大作。據講明旦要做風颱，這會兒店員也在問新來的客：「有下雨嗎？」我書看得有些累，竟在靠椅上喘了一會兒氣，但接落來又沈浸了進去。直到店員叩桌，提醒打烊，纔依依不捨地醒來，將書歸了位。

Hexo+NexT建站難關隨錄

刊於 2019-08-18 | 改於: 2019-09-13 | 屬於 隨筆 | | 閱覽:

§小引

想擁有一個徹底屬於自己的博客，已經很久了。前天，做了一場很長很長的夢後，我開始著手建設之。

我猶能記取，最後一回正經的編程，已是初中的事情了。當時電腦課有教VB，我上課學滿認真，落堂自家還繼續琢磨；復在百度VB吧圍觀大神，斗膽提問並得到回答：可以說充分地體驗了編程的樂趣。期末的作品，亦蒙先生表揚，說我「既達成了目標，還有意識設計了圖形介面」云。然而（é），這份編程愛並未延續落去。特別進了高二，我躊躇滿志地選了文科，也接受著能力偏差的心理暗示。故此從我指尖流出的代碼，邇來大抵是些臨時搜索、複製黏貼，以應付突發的疑難雜症者。「編程」之於我，遂與日而神聖，與日而不可及了。這份複雜的情感，更從編程開始，蔓延到許多的現代信息技術。

我到底是转厝了。

空气颇湿闷，呼吸起来却极爽快。耳边聒噪得很，却是亲切的乡音。行人在扶手道上向左向右乱靠，我也跟在后面做无规则运动。停车场多了扫描二维码计费的牌牌，车窗外则挂着一轮圆月。

我又感到新鲜，又感到熟悉。

以往不谏

刊於 2018-12-13 | 改於: 2018-12-13 | 屬於 隨筆 | | 閱覽:

《昨日青空》，看得一阵胃疼，打了四星。是打给谁的呢。

一直以来和友人勉励着“来者可追”，便只管埋首今天，不甚回头――哪怕瞥一眼——瞥一眼过去灰头土脸的自己。

偶尔会忍不住向友人追述起过去的时光。当然，故事里肯定要有友人的身影。友人似乎饶有兴致地倾听着。突然，“我都不记得了欸。”我便惶恐地收住嘴。不想人留下这样的印象——像自家厝里那台掉漆的文曲星一样，只能按下高保真，听它用沙哑的音色勉强复读着过去的事情。
