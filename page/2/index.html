天の川

十六暝晡煙共花

刊於 2018-05-23 | 改於: 2018-12-13 | 屬於 紀行 | | 閱覽:

正月十六暝晡，我共行李搬轉宿舍。宿舍固無別儂，我整理完，自家出門，去食日本菜。

經過旁邊其村，巷巷盡窄。厝裏捅出棍棍，野像會敲響對面檻門囝其玻璃。遐辰候天暗無偌𣆯，前兜俤蜀隻丈夫囝蒂甩了摔炮，嘻嘻笑䟛去。只爿伲囝哥掏𡅏㲺㲺花，邊兜儂圍蜀環相。盡価厝門兜放了盆盆，盆盆貯了土，插了有碗許蠻大其香燒，煙噗囫圇巷巷都是。只大其香，以前㑚敆廟門兜看見過。

起五更早掏書看

刊於 2017-12-11 | 改於: 2019-09-02 | 屬於 隨筆 | | 閱覽:

今旦我自家也𣍐仈蔣倛了，起五更早𡅏睏𣍐着，眠牀𡅏輾轉反側。遘此刻五點倛時候，肩頭罔變罔𣍐受，就爬起了。雖然講我肩頭平常就𣍐好受，時間𣆯去就習慣了；但是過了睏會着倛時間再去睏，復睏𣍐着，身體倛痛就會放大。

爬起無事做，𣍐愛覆桌𡅏，就討書掏去罔看。凊天遘六點固是暗，固有野価時間。
