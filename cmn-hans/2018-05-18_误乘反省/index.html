误乘反省

刊於 2018-05-18 | 改於: 2019-08-27 | 歸於 隨筆 | | 閱覽:

这次出行弄错了始发站，付出了不必要的金钱和精力代价。我本有很多次机会可以避免这一问题。现在将所有能想到的纠偏机会，同本应采取的正确措施，列在下方。

  1. 网上订票时，就应确认清楚网页上的起迄信息。而不应想当然地相信自己所定的与实际定下的相同。

  2. 应提前在自己的手机安装好火车票应用。这样在出现问题时可以立即措置。

  3. 提前抵站取票时，应再次确认起迄站名。

  4. 应当将取好的票，放在钱包最显眼的透明层里。这样就能有无数次确认的机会。

  5. 要在抵达车站与搭乘火车之间留下一个小时。这次■■完全信任我的安排，选择踩点抵达，而我没有很好地珍惜这份信任。

六点十分近二十分左右，我被检票人员阻拦，告知应抵北站。明知失期难以弥补，我仍然采取了以下不明智的行为：

  1. 联系■■立刻改向，同时自己也迅速回站搭上地铁。但是地铁的运行时间固定，很难出现奇迹。当时南北站之间差了十七个地铁站，我估计一站行二分钟，以为只需三十四分钟，想要赌一把，实际上三十四分钟后下车已经来不及，遑论实际上每站还要停靠一分钟。当时正确的做法是立刻网络改签。

  2. 在地铁上，仍然有机会弥补改签。但是我心烦意乱，没有平静地和■■沟通，只因身边吵闹，无法听清，就毫无耐心地挂断电话。这就失去了及时让■■协助改签的机会。我也没有重视一位朋友的意见。他建议我赶在开车前立刻改签，但我正焦躁着，没有再次上网验证得到的信息，就直接相信了「车开走后也能改签」的说法，一味地想要去火车北站赌一把。这便浪费了止损的机会。

  3. 着站后，被验证机器阻拦，要求改签。到改签窗口被告知，车已行则票废，不能改签。如果不是■■恰好找到我们，我们甚至拿不出再买第二日车票的钱。旅行前，本应针对特殊情况备足金额。

本次失误，反映出个人的缺陷：

  1. 我对关键处、细节处的把控能力不足。因为■■■■■■■■■■事务比较显眼，半个月来，我的工作重心都集中在这上面。此为我也为调查需要为朋友代购的东西付出了一点精力。但最重要的住行信息，我却想当然了。

  2. 我太过汲汲于证明自己「会做事计」，今回旅行也一定能安排好。这在某种意义上形成了一种浮躁的心态，让我以为自己面面俱到，却在意想不到的关键处疏失。

  3. 今月为旅行做的准备当中，也有一些可取之处。比如我懂得■■无路用的■■■■，活用时间去取票。又如，自南站下地铁欲返回北站时，一侧安检口堵塞，工作人员劝导大家前往另一口安检，无人相信其言，我便奔向另一口，抢得上地铁的先机。这也许体现出了我的应急反应能力，而我长久以来自认欠缺这方面的能力。但就最终情况来看，我的应急能力是浮于表面的。这份浮于表面的应急能力和我最终蒙受的损失相辅相成。我会偷闲取票，却不晓得多看票面一眼。我会想办法抢搭下一班地铁，却不晓得停下来把时间算清楚，来判断是不是要改签。

这场旅行，最终幸能成行。如果以后为团队安排交通，疏失的后果更是不容小觑。我尽量吸取教训。
